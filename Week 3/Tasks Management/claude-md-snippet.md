## Personal Operating System

This workspace runs a small Personal Operating System. Read the files below before answering planning or task questions.

**Files**
- `goals.md`: up to 3 quarterly goals. The source of strategic direction.
- `tasks.md`: current backlog, pipe-delimited, one task per line.

**Skills**
- `/daily-plan`: produces three priorities for today from `goals.md`, `tasks.md`, and today's calendar.

**Behaviour**
- When I ask what to work on today, run `/daily-plan`. Don't freestyle a list.
- Before editing `goals.md`, ask first. Goals change rarely.
